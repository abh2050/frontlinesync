Privacy Policy

Effective Date: August 9, 2025

1. Information We Collect

FrontLine Sync is designed with privacy in mind. We collect minimal information:

  • Local Storage Data: All your data (shifts, credentials, chat history) is stored locally in your browser
  • API Keys: Gemini API keys you configure are stored locally and never transmitted to our servers
  • Usage Analytics: We may collect anonymous usage statistics to improve the application

2. How We Use Your Information

  • Provide and maintain the FrontLine Sync service
  • Process AI chat requests through your configured Gemini API
  • Improve user experience and application functionality
  • Ensure compliance with industry standards

3. Data Storage and Security

  • Local Storage: All personal data is stored in your browser's local storage
  • No Server Storage: We do not store your personal information on our servers
  • HTTPS Encryption: All communications are encrypted in transit
  • API Security: Your Gemini API key is stored locally and used directly with Google's services

4. Third-Party Services

We integrate with:

  • Google Gemini AI: For AI chat functionality (subject to Google's privacy policy)
  • GitHub Pages: For hosting (subject to GitHub's privacy policy)

5. Your Rights

  • Access your data (all stored locally in your browser)
  • Delete your data (clear browser storage)
  • Export your data (browser developer tools)
  • Control API integrations (configure or remove API keys)

6. Industry Compliance

FrontLine Sync is designed to support HIPAA, GDPR, and other regulatory compliance through:

  • Local data storage (no server-side personal data)
  • User-controlled data retention
  • Secure communication protocols
  • Audit trail capabilities
